Graph

Diberikan graph diatas, kemudian akan di berikan warna pada setiap node nya.
syaratnya:
* gunakan warna sesedikit mungkin { biru, merah, kuning, hijau }
* node yang bertetangga tidak dibolehkan warnanya sama




output:

```
titik b = biru
titik c = merah
titik a = kuning
titik d = kuning
titik e = biru
titik f = merah
```
